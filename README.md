# ATS Resume Optimizer — Claude.ai Skill

A Claude.ai skill that transforms resumes into ATS-optimized, 
recruiter-ready documents targeting 100/100 ATS compatibility scores.

## Features
- Deep job description analysis & keyword extraction
- Job title alignment (keeps employment dates intact)
- Metric excavation to quantify achievements
- LinkedIn & portfolio ingestion for richer evidence
- ATS-compliant formatting (no tables, columns, or graphics)
- Full optimization report with keyword scores

## Installation
1. Download `ats-resume-optimizer.skill`
2. Open Claude.ai → Settings → Skills
3. Upload the `.skill` file
4. Done — skill auto-triggers when you share a resume + job description

## Usage
Just paste your resume and job description into Claude and say:
> "Optimize my resume for this job"
