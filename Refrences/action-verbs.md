## High-Impact Phrases to Use

Pair these with your action verbs and metrics for maximum ATS and recruiter impact:

- "resulting in..."
- "achieving a [X]% improvement in..."
- "reducing [metric] by [X]% within [timeframe]"
- "driving [X]% growth in [metric]"
- "saving [amount/time] annually"
- "across [X] markets / regions / countries"
- "for [X]+ users / customers / stakeholders"
- "ahead of [X]-week schedule"
- "within [budget/constraint]"

## Verbs to AVOID (Weak / Vague)

These appear frequently on resumes but add no signal — replace them:

❌ Responsible for → ✅ Managed / Led / Delivered
❌ Helped with → ✅ Contributed to / Supported / Enabled
❌ Worked on → ✅ Developed / Built / Executed
❌ Assisted in → ✅ Collaborated on / Accelerated / Improved
❌ Tasked with → ✅ Directed / Oversaw / Spearheaded
❌ Participated in → ✅ Drove / Contributed / Facilitated
❌ Was involved in → ✅ Led / Managed / Delivered
❌ Handled → ✅ Managed / Resolved / Administered
❌ Did → ✅ Executed / Produced / Delivered
❌ Made → ✅ Created / Built / Launched / Generated
